# MTI Smart Library Management System 📚

A professional Python-based CLI application for managing university library records, focusing on data persistence and automated reservation logic.

## ✨ Features
- **Automated Reservation:** Calculates return deadlines (14 days) automatically.
- **Data Persistence:** Uses JSON for permanent storage of book records.
- **Self-Healing Data:** Automatically repairs missing data fields during runtime.
- **Professional UI:** Utilizes the `Rich` library for organized terminal tables.

## 🛠️ Built With
- **Python 3.x**
- **JSON** (Database)
- **Rich Library** (Terminal UI)
